# Contributing

Thanks for improving Crop Guide Overlay.

## Development

1. Fork or branch from `main`.
2. Keep changes focused and do not commit `.build` output or local LaunchAgent state.
3. Build the app and package:

```bash
make clean
make app
make package
```

4. Test a short screen recording with the capture method affected by your change.
5. Explain the user-visible impact and validation in the pull request.

Use public issues for ordinary bugs. Follow [SECURITY.md](SECURITY.md) for vulnerabilities and [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for conduct expectations.
