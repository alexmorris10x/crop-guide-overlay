# Support

Use [GitHub Issues](https://github.com/alexmorris10x/crop-guide-overlay/issues) for reproducible bugs and focused feature requests. Include the macOS version, capture tool, build or release version, steps to reproduce, and the observed result.

This project is maintained on a best-effort basis with no guaranteed response time. Test the overlay with your exact capture workflow before a production recording.

Do not post credentials, private recordings, or vulnerability details in public issues. Report security problems through [SECURITY.md](SECURITY.md).
